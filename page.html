Hello, So you have heard of git, and you want to use it. Follow these steps.
  • For Windows:
    1. Download the git exe file
    2. Install the exe file
    3. Open git bash
    4. Create a folder with some stuff. For example create a web page like this one
    5. Cd into the the new folder, and create a repo.
      • cd new_folder
      • git init
    6. Now, lets add our web_page.html into the staging area.
      • git add web_page.html
    7. Next, commit the web_page.html
      • git commit -m "created a web page"
